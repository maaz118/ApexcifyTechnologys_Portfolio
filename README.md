# ApexcifyTechnologys_Portfolio

## Project Overview
This project is a personal portfolio website created using HTML and CSS.  
It was developed during my Web Development Internship at ApexcifyTechnologys.
The portfolio website showcases my skills, projects and contact information in a clean and responsive layout.

## Technologies Used
- HTML5
- CSS3
- JavaScript 

## Features
- Responsive website layout
- About Me section
- Skills section
- Projects showcase
- Contact information
- Hover effects for interactive design

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your web browser
